# 📱 دليل تحويل "مستودع المدى" إلى تطبيق أندرويد APK وتشغيله بدون إنترنت

لقد قمنا بتجهيز وتحديث كافة الملفات المطلوبة لجعل التطبيق جاهزاً تماماً للعمل كـ **تطبيق مستقل (PWA)** أو تحويله إلى **تطبيق أندرويد APK أصلي** يعمل **بشكل كامل أوفلاين (بدون إنترنت)**.

---

## ⚠️ سبب مشكلة "الصفحة غير موجودة" (Error 404) التي ظهرت لك
رابط النسخة المشتركة (Shared URL) يظهر لك خطأ 404 لسبب بسيط جداً:
* **الحل:** يجب عليك الضغط على زر **"Deploy" (نشر)** أو زر **"Share" (مشاركة)** الموجود في أعلى يمين واجهة Google AI Studio لتفعيل الرابط ونشر التحديثات الأخيرة إليه. بعد تفعيله، سيعمل الرابط فوراً وبدون أي مشاكل!

---

## 🛠️ كيف تحصل على تطبيق APK يعمل بدون إنترنت؟ (3 طرق مضمونة وأكيدة)

### 🌟 الطريقة الأولى: استخدام موقع PWABuilder (سهلة، سريعة، ومجانية 100%)
هذه هي أسهل وأسرع طريقة للحصول على ملف **APK** جاهز للتثبيت دون الحاجة لكتابة أي كود أو تثبيت برامج على كمبيوترك:

1. اضغط على زر **"Deploy" (نشر)** في Google AI Studio لتفعيل الرابط.
2. انسخ رابط تطبيقك المستقر (Shared App URL) الذي ينتهي بـ `europe-west1.run.app`.
3. افتح موقع: **[PWABuilder](https://www.pwabuilder.com/)**
4. ضع رابط تطبيقك في الخانة المخصصة واضغط **Start**.
5. سيقوم الموقع بفحص التطبيق وسيظهر لك **علامات نجاح خضراء بالكامل** (لأننا قمنا ببرمجة ملف `manifest.json` و `sw.js` وربطهما بنجاح في الكود!).
6. اضغط على زر **Package for Stores**.
7. اختر **Android** واضغط على زر **Generate Package** ثم قم بتحميل ملف الـ **APK** وتثبيته مباشرة على جوالك!

---

### 📱 الطريقة الثانية: التثبيت المباشر من المتصفح (PWA) - تعمل بالكامل بدون إنترنت!
بما أننا قمنا ببرمجة **Service Worker** متكامل، يمكنك الآن استخدام التطبيق بدون إنترنت مباشرة من الجوال:

1. افتح رابط التطبيق المباشر (المطور) أو المستقر في متصفح **Chrome** على جوال الأندرويد.
2. ستظهر لك لوحة زرقاء أسفل الشاشة أو في الصفحة الرئيسية تطلب منك **"تثبيت التطبيق على جوالك"**.
3. إذا لم تظهر، اضغط على زر **الثلاث نقاط (⋮)** في أعلى يسار متصفح Chrome.
4. اختر **"إضافة إلى الشاشة الرئيسية" (Add to Home screen)** أو **"تثبيت التطبيق" (Install app)**.
5. سيظهر التطبيق كأيقونة على شاشتك الرئيسية، وعند فتحه **سيعمل بسرعة فائقة وبدون الحاجة لإنترنت نهائياً!**

---

### 💻 الطريقة الثالثة: بناء تطبيق APK محلياً على جهازك باستخدام Capacitor
إذا كنت تريد بناء ملف الـ APK بنفسك على جهاز الكمبيوتر الخاص بك، إليك الخطوات بالتفصيل والمفهوم البسيط:

#### ما هو Capacitor؟
هو أداة مجانية تقوم بإنشاء "غلاف" (Wrapper) يضم ملفات الويب الخاصة بالتطبيق (React) ويقوم بتحويلها وتشغيلها داخل تطبيق أندرويد حقيقي وكأنها تطبيق أصلي يعمل أوفلاين تماماً.

#### خطوات البناء على كمبيوترك:
1. قم بتحميل مشروعك كملف **ZIP** من قائمة الإعدادات (Settings -> Export ZIP) في AI Studio وقم بفك الضغط عنه.
2. افتح موجه الأوامر (Terminal / Command Prompt) داخل مجلد المشروع الذي قمت بفك ضغطه.
3. قم بتثبيت Capacitor في المشروع عبر تشغيل الأوامر التالية:
   ```bash
   npm install @capacitor/core @capacitor/cli
   ```
4. قم بتهيئة Capacitor في المشروع (سيطلب منك اسم التطبيق ومعرّف الحزمة):
   ```bash
   npx cap init "مستودع المدى" "com.almadawarehouse.app" --web-dir=dist
   ```
5. قم بتثبيت منصة الأندرويد:
   ```bash
   npm install @capacitor/android
   npx cap add android
   ```
6. قم ببناء ملفات الويب الخاصة بالتطبيق:
   ```bash
   npm run build
   ```
7. قم بنسخ ملفات البناء إلى مجلد الأندرويد:
   ```bash
   npx cap sync
   ```
8. لفتح المشروع وبناء الـ APK، قم بتشغيل الأمر التالي لفتحه في برنامج **Android Studio**:
   ```bash
   npx cap open android
   ```
9. من داخل برنامج Android Studio، اضغط على **Build -> Build Bundle(s) / APK(s) -> Build APK(s)**.
10. مبروك! ستحصل على ملف الـ APK الخاص بك جاهزاً وبأيقونة التطبيق الرسمية التي صممناها لك!
